# Task-by-task classification

Source ranges = the handwritten note (`reference/fufi_summer_math_tasks.jpg`), validated.
Each task classified from the page photos by **type** (expression / textual) and **difficulty**.

**Difficulty rules used:**
- **HARD** — long/exhausting multi-step calculation · textual word problem (esp. with а)/б)/в) sub-parts) · **unit conversion** (t→mg, km→mm, dm²→cm², m³→cm³, area/volume units) · setting up an equation / geometry reasoning.
- **EASY** — expression that rationalises nicely / cuts to ×10·×100·×1000 (or ÷) · round numbers · quick single-step.
- **MEDIUM** — moderate calculation, or a short simple word problem without heavy bullets.
